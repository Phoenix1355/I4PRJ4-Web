Module: Pages/Index

The main page displaying the rides. The client must be logged in to see this page, because of the 'auth' middleware.

Data

Name Type Default value Description
items Array - The list of retrieved items
errorMessage String - The latest error message
interval function - THe interval loop used to call retrieve event

Computed

Name Type Description
name String Returns the name of the current state account
loggedIn Bool Return the login status from the current state

Events

Name Payload Type Description
logout The logout method dispatching 'logout' to the store
retrieve Array retrieves open rides from backend
